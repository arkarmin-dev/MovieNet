Captain Colter Stevens (Jake Gyllenhaal) သည် အမေရိကန်စစ်တပ် ဂျက်လေယာဉ်မှူးတစ်ဦးဖြစ်ပြီး သူသည် ဂျက်လေယာဉ်မှူးတစ်ဦးဖြစ်သော်လည်း သူသည် ကိုယ်ပိုင်ခန္ဓာကိုယ်မှာမဟုတ်ဘဲ သင်္ကာပတေယ့်ခန္ဓာကိုယ်ဖြစ်သော Sean Fentress အဖြစ် အခြားသူ၏ခန္ဓာကိုယ်အတွင်းရောက်ရှိသွားသည်ကိုတွေ့ရသည်။ ယင်း Chicago နိုင်ငံရှိ ချီကာဂို ကွန်ပျူတာရထားပေါ်တွင် ဗုံးတစ်လုံးပေါက်ကွဲခဲ့သည်မှာ သူတို့က စုံစမ်းစစ်ဆေးမှုတစ်ခုရောက်ရှိလာသည့်အစီအစဉ်တစ်ခုဖြစ်ကြောင်းတွေ့ရှိခဲ့သည်။ စတီဗင်သည် လွန်ခဲ့သော ရှစ်မိနစ်၏ အခြားသူ၏ဘဝကိုခံစားခွင့်ပေးသည့် "Source Code" ခေါ်အစီအစဉ်တစ်ခုတွင် ပါဝင်နေသည်ကို တွေ့ရှိခဲ့သည်။ အခြားသူ၏နောက်ဆုံးရှစ်မိနစ်ကို အကြိမ်ကြိမ်ပြန်လည်ဖြတ်သန်းသည်အခါ စတီဗင်သည် ဗုံးခွဲသူကို ရှာဖွေနိုင်ရန်နှင့် များစွာသော အသက်များကို ကယ်တင်နိုင်ရန် clues တွေကို ဆက်စပ်ဖို့ လိုအပ်သည်။
